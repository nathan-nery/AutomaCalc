git init
git remote add origin linkkk
git clone linkkk

Se pedir senha e não aceitar o login e senha normais, criar um Token em:
GitHub > Settings > Developer settings (lá embaixo) > Personal access Tokens > Tokens > Generate new Token

Para commitar:

git add .
git commit -m "nome do commit"
git push origin main

Mudança de Branch:

#aqui você cria e entra na nova branch automaticamente
git checkout -b NovaBranch
git push origin NovaBranch

Merge de branchs:

#sai da branch criada e volta pra main
git checkout main
git merge NovaBranch
git push origin main
